# Week 01. CLI programming

## Open KTB CLI
- 카카오부트캠프 관련 웹사이트 자동 실행 CLI

### Ideas
- 강의 수강 시 매일 열람하는 카테부 관련 웹사이트 한 번에 오픈
- 디폴드 웹사이트 : 출석QR, ZEP, ZOOM
- 선택 옵션 : Notion, GitHub

### How to use
- open_ktb 입력 : 디폴트 웹사이트(3) 실행
- open_ktb notion git100 입력 : 선택 웹사이트 추가 실행
- 총 실행된 웹사이트 개수 만큼 print 출력 : "Kakao Bootcamp {num} website is opening ..."
